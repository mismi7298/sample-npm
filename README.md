# sample-debug-chalk

Sample npm project using [debug](https://www.npmjs.com/package/debug) 4.4.3 and [chalk](https://www.npmjs.com/package/chalk) 5.6.2.

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

To see debug output:

```bash
npm run debug
```

Or with a specific namespace:

```bash
DEBUG=app:* node index.js
```
